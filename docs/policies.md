# 정책 문서 (초안)

공개 사이트 게시 전 검토가 필요한 정책 초안이다. 법적 효력을 갖기 전 법무
검토를 거쳐야 한다.

## 개인정보 처리방침 (초안)

- 본 서비스는 회원가입·로그인이 없으며, 서버에 개인 식별 정보를 저장하지
  않는다.
- 이용자의 관심종목·보유 수량·평단·메모는 이용자 브라우저의 `localStorage`에만
  저장되며 서버로 전송되지 않는다.
- 서버는 공개 데이터(종목, 공시, 뉴스, 분석 결과)만 저장한다.

## 쿠키 정책 (초안)

- 현재 서비스는 쿠키를 사용하지 않는다. 관심종목 저장에는 쿠키가 아닌
  브라우저 `localStorage`를 사용한다.
- 향후 analytics 또는 광고를 도입하면 해당 쿠키 사용 내역을 이 문서와 쿠키
  동의 UI에 반영한다.

## 광고 정책 (초안)

- 현재 광고 네트워크는 연동돼 있지 않다.
- 광고 도입 시: 광고와 편집 콘텐츠를 명확히 구분하고, 광고 제공자가 수집하는
  데이터 범위를 쿠키 정책에 명시한다.

## 면책 (disclaimer)

- 본 서비스는 OpenDART 공시와 Naver 뉴스를 수집·요약해 제공하는 정보
  서비스이며, 투자 자문·권유가 아니다. 투자 판단과 그 결과의 책임은 이용자
  본인에게 있다. (사이트 footer에 상시 표기.)
